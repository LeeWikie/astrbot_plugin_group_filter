# 群聊过滤器插件

一个 AstrBot 插件，用于监听指定群聊消息，通过大模型判断内容是否违规，并对违规消息执行撤回和警告。

## 功能特性

- 📋 **群聊白名单管理**：通过配置文件维护需要监控的群聊ID列表
- 🤖 **AI 内容审核**：调用大模型判断消息是否违规
- 🚫 **自动撤回**：对违规消息执行撤回操作
- ⚠️ **警告通知**：@违规成员并发送警告消息
- 🌐 **WebUI 配置**：支持通过 AstrBot WebUI 管理配置

## 安装

1. 将插件目录放置于 AstrBot 的 `data/plugins` 下
2. 重启 AstrBot 或在 WebUI 中启用插件
3. 在 WebUI 的"插件配置"页面配置参数

## 配置说明

### monitor_groups
监控的群聊ID白名单（多个ID用英文逗号分隔）
示例：`754697401,573545488`

### filter_prompt
AI判断消息的提示词
默认：`你是一个群聊管理员，请判断以下消息是否违规。违规内容包括但不限于：广告、恶意刷屏、人身攻击、政治敏感。如果消息违规，请输出<filtered>；如果消息正常，请输出<none>。只输出标签，不要有其他内容。`

### violation_message
检测到违规时的警告消息
默认：`⚠️ 检测到违规内容，您的消息已被撤回。请遵守群规，文明交流。`

## 工作流程

1. 插件监听所有群聊消息
2. 检查消息群ID是否在白名单中
3. 调用大模型判断消息内容
4. 若模型返回 `<filtered>`：
   - 撤回该消息
   - @消息发送者并发送警告
5. 若返回 `<none>` 或无违规，不作处理

## 注意事项

- 机器人账号必须在目标群中拥有**撤回消息**的权限（通常是管理员或群主）
- 每条消息都调用大模型，可能会产生较高延迟和API费用
- 建议监控AI调用频率，避免超出配额

## 依赖

本插件主要使用 AstrBot 内置功能，无需额外依赖。
